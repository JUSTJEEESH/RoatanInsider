Roatán Insider

Complete Content Update Guide

The 3-Step Pattern (Same for Everything)

1 Export — In Terminal:

cd ~/Documents/RoatanInsider
python3 tools/export_all.py

This creates CSV files for everything in the app.

2 Edit — Open the CSV(s) you want to change in Google Sheets, Numbers, or Excel. Save when done.

If using Google Sheets: File → Download → CSV, then in Terminal:

mv ~/Downloads/FILENAME.csv ~/Documents/RoatanInsider/FILENAME.csv

3 Import & Upload — In Terminal:

python3 tools/import_all.py --bump-version

Then upload the changed JSON files + manifest.json to Supabase → Storage → app-data bucket.

All Terminal Commands (Copy & Paste)

Export all content to CSVs:

cd ~/Documents/RoatanInsider
python3 tools/export_all.py

Move edited CSV back from Downloads:

mv ~/Downloads/FILENAME.csv ~/Documents/RoatanInsider/FILENAME.csv

Import all CSVs and bump version:

cd ~/Documents/RoatanInsider
python3 tools/import_all.py --bump-version

Preview without writing (dry run):

python3 tools/import_all.py --dry-run

Pull latest code updates:

cd ~/Documents/RoatanInsider
git pull origin claude/curate-business-list-r77Mw

Testing Changes

  1. Upload files to Supabase
  2. Fully close the app on your phone (swipe up in app switcher)
  3. Wait 5 seconds
  4. Reopen the app
  5. Your changes should appear

If changes don't appear, make sure you uploaded manifest.json — the app only downloads new data when the manifest version increases.
