Roatán Insider

Business Update Guide — Daily Workflow

1 Get the Latest CSV

Open Terminal (Cmd + Space → type Terminal → Enter)

Paste these two lines and hit Enter:

cd ~/Documents/RoatanInsider
python3 tools/export_csv.py

You'll see: ✅ Exported businesses to businesses_edit.csv

2 Edit in Google Sheets

Open: Go to sheets.google.com

Import: File → Import → Upload → drag in businesses_edit.csv from Finder (Documents → RoatanInsider)

Choose: Replace spreadsheet → Import

Common edits:

To do this...Change this column
Edit a name, description, tipName, Description, Insider Tip
Update contact infoPhone, WhatsApp, Website, Instagram
Feature a businessFeatured → yes
Make it an Insider PickInsider Pick → yes
Remove a businessStatus → inactive
Add a businessNew row with ID (e.g. b202), Name, Category, Area, Status = active
Change hoursHours Text (e.g. "Daily 11am–10pm")

Valid categories:

Eat · Drink · Dive · Tours · Shop · Stay · Rentals · Transport · Beaches · Nightlife

Valid areas:

West Bay · West End · Sandy Bay · Coxen Hole · Flowers Bay · French Harbour · Oak Ridge · Punta Gorda · Port Royal · Camp Bay

When done: File → Download → Comma Separated Values (.csv)

3 Move the File Back

In Terminal, paste:

mv ~/Downloads/businesses_edit.csv ~/Documents/RoatanInsider/businesses_edit.csv

If it says "No such file", your download may have a different name like businesses_edit (1).csv. Check your Downloads folder.

4 Convert to JSON

In Terminal, paste:

cd ~/Documents/RoatanInsider
python3 tools/import_csv.py --bump-version

You'll see a summary with business counts. Make sure the numbers look right.

5 Upload to Supabase

Go to Supabase Dashboard → Storage → app-data

Upload #1 — businesses.json

  1. Delete the old businesses.json (three dots → Delete)
  2. Click Upload files
  3. Navigate to: Documents → RoatanInsider → RoatanInsider → Data → businesses.json

Upload #2 — manifest.json

  1. Delete the old manifest.json (three dots → Delete)
  2. Click Upload files
  3. Navigate to: Documents → RoatanInsider → manifest.json
⚠️ businesses.json is inside: Documents → RoatanInsider → RoatanInsider → Data
⚠️ manifest.json is inside: Documents → RoatanInsider (root folder)

6 Test

  1. Fully close the app (swipe up in app switcher)
  2. Reopen the app
  3. Your changes should appear

Quick Reference

✅ Always upload these to Supabase:
• businesses.json — from RoatanInsider → RoatanInsider → Data
• manifest.json — from RoatanInsider root folder
❌ Never upload these:
• businesses_edit.csv — editing file only
• Anything from the tools/ folder
